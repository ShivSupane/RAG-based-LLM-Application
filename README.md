# RAG LLM — Chat With Your Documents

Upload `.txt` files, enter your free Groq API key, and chat with your documents using LLaMA 3.

## How to use
1. Get a free API key from [console.groq.com](https://console.groq.com)
2. Upload your `.txt` documents
3. Click **Build Index**
4. Start asking questions!

## Tech Stack
- **LLM**: Groq (LLaMA 3.1 8B) — free and fast
- **Embeddings**: HuggingFace all-MiniLM-L6-v2 — runs locally
- **Vector Store**: FAISS
- **Framework**: LangChain + Gradio
